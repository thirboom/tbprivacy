Privacy Policy

Last updated: June 10, 2026

This application is a personal-use tool for automating video downloads, editing, and posting to a Facebook Page. It is not a public service and is not distributed to third parties.

1. Information We Collect

This application does not collect, store, or transmit any personal data from users. The only credentials used are a Facebook Page Access Token stored locally on the device of the application owner.

2. How We Use Information

The Facebook Page Access Token is used solely to publish video content to the owner's Facebook Page via the Facebook Graph API. No data is shared with any third party other than Facebook's own API endpoints.

3. Data Storage

All configuration data (including access tokens) is stored locally on the owner's personal computer and is never uploaded to any external server operated by this application.

4. Third-Party Services

This application interacts with the following third-party services:

  • Facebook Graph API — for publishing videos to a Facebook Page
  • YouTube (yt-dlp) — for downloading publicly available video content

Use of these services is subject to their respective privacy policies and terms of service.

5. Contact

This is a private application. If you have questions, please contact the application owner directly.
